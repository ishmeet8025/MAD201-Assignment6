## Overview
News Reader App fetches technology news from the Inshorts API, displays them in a list,
allows viewing details and saving favorites locally using AsyncStorage.

## Run
1. Install Expo CLI: `npm install -g expo-cli`
2. Install dependencies: `npm install`
3. Start: `npm start`
4. Use Expo Go or emulator: `npm run android` or `npm run ios` (mac)
